## 1. Introduction
The Mental Health Journal Web App is a personal wellbeing tracker
that allows users to log daily moods, reflect on emotions,
upload media and visualize mental health trends.

---

## 2. Objectives
- Promote mental wellbeing awareness
- Help users track emotional patterns
- Provide insights through analytics
- Encourage self-reflection

---

## 3. Features
- User authentication (login & register)
- Daily mood entries
- Upload photos/audio
- Weekly reflections
- Mood analytics
- Streak tracking
- PDF export

---

## 4. System Architecture
**Frontend**
- HTML
- CSS
- JavaScript
- Chart.js

**Backend**
- Flask
- SQLAlchemy ORM
- SQLite database

---

## 5. Database Design

### Tables
- users
- entries
- entry_media
- weekly_reflections

Each table is connected using foreign keys.

---

## 6. Application Flow
1. User registers
2. Logs in
3. Creates journal entry
4. Uploads media
5. Views dashboard analytics
6. Exports reports

---

## 7. Security

- Protected routes
